# Chatroom
Angular的小型聊天室demo

基本没有功能

## Install dependencies
```
npm install
```
## Run
To start a webserver with source maps do:
```
npm start
```
## Testing
```
npm run test
```
## Create production bundle
```
npm run build
```
## Notes
- 在tslint.json中设置前缀规则，默认是app，意味着component的标签都是`<app-xxx></app-xxx>`
- component的局部样式（貌似？）既可以用scss也可以用普通css
